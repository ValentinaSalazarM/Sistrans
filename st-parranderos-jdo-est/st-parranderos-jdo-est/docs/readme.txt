Pasos para instalar la aplicación AforoAndes

-> Correr la clase InterfazAppAforoAndes
-> Registrarse o iniciar sesión en el botón superior izquierdo con el fin de poder probar las funcionalidades exclusivas de administradores o administradores de locales.
-> Es posible iniciar sesión con los siguientes datos:

	ADMINISTRADOR (Privilegios para todas las consultas):

	* Identificación: 3543016364021851          Contraseña: #618
	* Identificación: 3550856365084741          Contraseña: #b8c
	* Identificación: 4913646598441655          Contraseña: #085

-> La totalidad de los requerimientos de consulta se encuentran en el item RequerimientosConsulta del Menú dependiendo los parámetros que requiere cada uno de éstos. En el caso de requerir la especificación de ventanas de tiempo para acotar la búsqueda, se habilitan calendarios y se permite ingresar horas. Si se trata de un rango de fechas (u horas) se debe hacer coincidir la fechaInicial (horaInicial) con la fechaFinal (horaFinal). Con algunos requerimientos se exige una horaInicial y con otros no de acuerdo al enunciado.

-> Al adicionar un ascensor no puede colocarse una capacidad normal puesto que ésta, de acuerdo a las normativas vigentes, es igual a 2.
-> Al adicionar un nuevo registro de carnet la hora de salida queda en NULL en la base de datos. Para actualizarla, se exige la misma información de adición. El proceso puede ser tardado puesto que se hacen múltiples verificaciones.
-> Los test del paquete "InterfazTest" prueban funcionalidades vinculadas al ingreso de datos por interfaz, por lo que por cada prueba se despliega el componente gráfico correspondiente. Existe cierto tiempo máximo permitido para colocar los datos especificados en el comentario del fragmento de código respectivo. Posterior a este tiempo el test indica que se ha generado un fallo. Algunos tiempos son extensos pero siempre terminan.
